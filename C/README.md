#  C Programming Exercises

This directory contains various C programs and mini-projects developed during my **Computer Engineering studies at METU NCC**.

The projects cover **fundamental and intermediate topics** in C programming including data structures, algorithms, file processing, and system simulations.

---

##  Relevant Courses
- CNG 140 – C Programming  
- CNG 213 – Data Structures  
- CNG 315 – Algorithms  

---

##  Featured Projects
These projects demonstrate stronger understanding of core computer science concepts:

- **AVL Fishdom Analysis**  
  AVL Tree implementation used to analyze fish population data while maintaining balanced search performance.

- **Social Network**  
  Graph-based social network simulation using **DFS traversal** to analyze connections between users.

- **Student Book Borrowing Management**  
  Hash table implementation with **linear probing, quadratic probing, and double hashing** for efficient record management.
